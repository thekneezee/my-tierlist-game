# [cite_start]Welcome to the Custom Tier List & Spin Wheel Game! [cite: 1]

[cite_start]Here is how you play: [cite: 2]

1. **Upload Your Contenders:** Click the Start Game button. [cite_start]A window will pop up asking you to select files from your computer. [cite: 3] [cite_start]Select all the images (like movie posters, album covers, or characters) you want to rank. [cite: 4] [cite_start]They will instantly appear in the "Space for the Images" pool at the bottom. [cite: 5]
2. **Spin to Win (or Lose):** Click Spin the Wheel. [cite_start]This will take you to the wheel screen, which has automatically loaded all your images into slices. [cite: 6] [cite_start]Click the center Spin button and watch it go! [cite: 7]
3. **See the Result:** Once the wheel stops, a popup will announce the selected image. [cite_start]Click Back to return to the main board. [cite: 8]
4. **Rank It:** Find the image that was just selected in the bottom pool. [cite_start]Click, drag, and drop it into the tier you think it deserves (S, A, B, C, or D). [cite: 9]
5. [cite_start]**Keep Going:** Repeat the process by clicking Spin the Wheel again until the bottom pool is completely empty and your Tier List is finished! [cite: 10]
6. [cite_start]**Start Over:** Click Exit Game if you want to wipe the board clean, clear the images, and start a brand-new list. [cite: 11]
