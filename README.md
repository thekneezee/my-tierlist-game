# Welcome to the Custom Tier List & Spin Wheel Game! 

Here is how you play: 

1. **Upload Your Contenders:** Click the Start Game button. A window will pop up asking you to select files from your computer. Select all the images (like movie posters, album covers, or characters) you want to rank. They will instantly appear in the "Space for the Images" pool at the bottom. 
2. **Spin to Win (or Lose):** Click Spin the Wheel. This will take you to the wheel screen, which has automatically loaded all your images into slices. Click the center Spin button and watch it go! 
3. **See the Result:** Once the wheel stops, a popup will announce the selected image. Click Back to return to the main board. 
4. **Rank It:** Find the image that was just selected in the bottom pool. Click, drag, and drop it into the tier you think it deserves (S, A, B, C, or D). 
5. **Keep Going:** Repeat the process by clicking Spin the Wheel again until the bottom pool is completely empty and your Tier List is finished!
6. **Start Over:** Click Exit Game if you want to wipe the board clean, clear the images, and start a brand-new list.
